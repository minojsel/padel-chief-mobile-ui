UX Flow
1
Sport · Name · Format · Date · Visibility
Select sport, enter a name, set date & time, choose public or private, then pick a format. Next enables only when name and format are filled.
2
Format Settings
Americano: Rotation → Rounds → Points. Traditional: Format Preset + Structure + Grand Final. Both paths share a Courts | Players row, then a live time estimate below.
3
Advanced (optional)
Registration deadline, location, description, score input permissions, rest time. Americano: Championship Rounds + Tiebreaker rules. Traditional: Consolation format, Winners Final override, max games cap.
Format Mapping
Americano family (Rotating/Team/Mixed): points accumulate per rally; partners/opponents rotate. Supported for both Padel & Pickleball.

Traditional: bracket tournament with fixed pairs, set/game scoring. Padel only.
