UX Flow
1
Default Browse
All tournaments sorted by date. Sport toggle and status chips are always visible for quick filtering. Tap search or tune to go further.
2
Search Opens
The title row slides out, a search bar slides in. The full tournament list remains visible. The tune icon stays accessible — filters can still be applied. Cancel returns to default.
3
Search + Filters
Typing narrows results by name. The sport toggle and status chips compound with the query — e.g. "Open" with Padel selected shows only Padel tournaments with "Open" in the name.
4
Sort & Format Sheet
The tune button works in any state — including while searching. Format and sort refine results further without disrupting the search query or active chips.
5
No Results
If the combined filters + query return nothing, a contextual empty state appears. The chips remain accessible so the user can widen their filter without cancelling their search.
Design Rule
Search, sport, status, format and sort are additive — they all apply simultaneously. The result count in the header always reflects the combined intersection, so the user always knows how many results their current state produces.
